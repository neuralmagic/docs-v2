# Installation

This page provides a concise installation guide for NeuralMagic's DeepSparse, SparseML, and SparseZoo, catering to different AI pathways such as generative AI, object detection, image classification, and natural language processing (NLP). Each section includes a quick install command and links to detailed documentation for further exploration.

## Prerequisites

Before installation, ensure your system meets the requirements for [DeepSparse](./deepsparse#prerequisites), [SparseML](./sparseml#prerequisites), and [SparseZoo](./sparsezoo#prerequisites).

## Task-Specific Installation

### Generative AI

Install DeepSparse, SparseML, and SparseZoo with generative AI support:

<VersionInjector targetVersion="==VERSION" prepend="==">
```bash
pip install deepsparse[llm,server]==VERSION sparseml[llm]==VERSION
```
</VersionInjector>

### Object Detection

Install DeepSparse, SparseML, and SparseZoo with object detection support:

<VersionInjector targetVersion="==VERSION" prepend="==">
    ```bash
    pip install deepsparse[yolov8,server]==VERSION sparseml[yolov8]==VERSION
    ```
</VersionInjector>

### Image Classification

Install DeepSparse, SparseML, and SparseZoo with image classification support:

### Natural Language Processing
