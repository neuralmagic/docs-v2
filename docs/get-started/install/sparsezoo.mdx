# Installing SparseZoo

Welcome to the SparseZoo installation guide. Here, you'll find instructions for various installation methods including PyPI and advanced options via GitHub.

Generally, you will not need to install SparseZoo directly.
Instead, you can install the SparseML or DeepSparse packages, which includes SparseZoo as a dependency.
However, if you wish to install SparseZoo directly, you can utilize the information below.

:::note
The installation commands correspond to the version of SparseZoo you're viewing in the documentation.
:::

## Prerequisites

### Hardware Requirements

- None

### Software Requirements

- OS
  - Linux is supported and is tested across Ubuntu, CentOS, and RedHat distributions.
  - MacOS is currently in beta.
  - Windows is not supported at this time. However, Windows users can install DeepSparse via Docker or a virtual machine.
- Python 3.8-3.11
- ONNX versions 1.5.0-1.15.0, ONNX opset version 11 or higher.

## Installation

### PyPI

To install the current version of SparseZoo via PyPI, use the following command:

<VersionInjector targetProduct="sparsezoo" targetVersion="==VERSION" prepend="==">
    ```bash
    pip install sparsezoo==VERSION
    ```
</VersionInjector>

### GitHub

For advanced users, installation from source code is available using the following command:

<VersionInjector targetProduct="sparsezoo" prepend="v" currentTag="main" ignoreNightly={true}>
    ```bash
    pip install git+https://github.com/neuralmagic/sparsezoo.git@VERSION
    ```
</VersionInjector>

Alternatively, install from a locally cloned repository with these commands:

<VersionInjector targetProduct="sparsezoo" prepend="tags/v" currentTag="main" ignoreNightly={true}>
    ```bash
    git clone https://github.com/neuralmagic/sparsezoo.git
    cd sparsezoo
    git checkout VERSION
    pip install -e .
    ```
</VersionInjector>

## Specialized Installations

### Development

For development purposes, this extra provides support for development tools.

<VersionInjector targetProduct="sparsezoo[dev]" targetVersion="==VERSION" prepend="==">
    ```bash
    pip install sparsezoo[dev]==VERSION
    ```
