# Installing [PRODUCT]

Welcome to the [PRODUCT] installation guide. Here, you'll find instructions for various installation methods including PyPI, Docker, and advanced options via GitHub.

:::note
The installation commands correspond to the version of [PRODUCT] you're viewing in the documentation.
:::

## Prerequisites

### Hardware Requirements
- Example: 4 GB RAM (Minimum)
- Example: 2 GHz Dual-core Processor

### Software Requirements
- Example: Python 3.6 or later
- Example: Compatible Operating Systems (e.g., Windows 10, Ubuntu 20.04)

## Installation via PyPI

To install the current version of [PRODUCT] via PyPI, utilize the following command:

<VersionInjector targetProduct="product">
```bash
pip install product==VERSION
```
</VersionInjector>

## Installation via Docker

To install the current version of [PRODUCT] via Docker, utilize the following commands:

<VersionInjector targetProduct="product" prepend="v" currentTag="latest">
```bash
docker pull ghcr.io/neuralmagic/product:VERSION
docker run -it ghcr.io/neuralmagic/product:VERSION
```
</VersionInjector>

## Advanced Installation via GitHub

To install the current version of [PRODUCT] via GitHub, utilize the following commands:

<VersionInjector targetProduct="product" prepend="v" currentTag="main">
```bash
pip install git+https://github.com/neuralmagic/product.git@VERSION
```
</VersionInjector>

Additionally, the following commands install via GitHub, but from a locally cloned repository:

<VersionInjector targetProduct="product" prepend="tags/v" currentTag="main" ignoreNightly={true}>
```bash
git clone https://github.com/neuralmagic/product.git
cd product
git checkout VERSION
pip install -e .
```
</VersionInjector>

## Installation for Specific Use Cases

Install additional features using extras:

<VersionInjector targetProduct="product">
```bash
pip install product==VERSION[feature]
```
</VersionInjector>

Example for development:

<VersionInjector targetProduct="product">
```bash
pip install product==VERSION[dev]
```
</VersionInjector>
